
  # Card Battle Game UI Design

  This is a code bundle for Card Battle Game UI Design. The original project is available at https://www.figma.com/design/bZf9InWwFuNleH1FD0jtBc/Card-Battle-Game-UI-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Desktop app (Electron)

  - `npm run desktop:dev` — run Vite + Electron in development mode
  - `npm run desktop:build` — build unpacked desktop app via `electron-builder --dir`
  - `npm run desktop:dist` — build desktop installers/packages

  Icons are stored in `build/icons/`:
  - `icon.png` (base)
  - `icon.ico` (Windows)
  - `icon.icns` (macOS)
